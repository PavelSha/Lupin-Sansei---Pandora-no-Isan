# ROM Map

* Заголовок
  * `0x000000-0x00000F` - iNES ROM Header Data (см. файл `header.bin`)
* Банк 00, страница 1
  * `0x000010-0x00048F` - Список всех структур [BackgroundScreenInfo](terminology.md#Структура-BackgroundScreenInfo) по заднему фону
  * `0x000490-0x00058B` - Таблица по комнатам, каждый элемент которого представляет собой адрес таблицы индексов из предыдущего списка структур BackgroundScreenInfo
  * `0x00058C-0x0007BA` - Список таблиц индексов структур BackgroundScreenInfo в рамках отдельной комнаты
  * `0x0007BB-0x0007E5` - Список таблиц с номерами банков CHR ROM для заднего фона
  * `0x0007EB-0x00080F` - Не используется
  * `0x000810-0x00140F` - Индексы по экранам заднего фона, уровень 3.0, все экраны (кроме водных)
  * `0x001410-0x00178F` - Таблица атрибутов заднего фона, уровень 3.0, все экраны (кроме водных)
  * `0x001790-0x00195F` - Массив метатайлов, уровень 3.0
  * `0x001960-0x001A8F` - Не используется
  * `0x001A90-0x001D9F` - Таблица атрибутов заднего фона, уровни 3.1-3.4, все экраны (кроме водных)
  * `0x001CC0-0x001D2F` - Таблица атрибутов заднего фона, уровень 3.0, все водные экраны
  * `0x001D30-0x001D9F` - Таблица атрибутов заднего фона, уровни 3.1-3.4, все водные экраны
  * `0x001DA0-0x001F9F` - Массив метатайлов, уровни 3.1-3.4 и водные экраны уровней 3.0-3.4
  * `0x001FA0-0x00200F` - Не используется
* Банк 00, страница 2
  * `0x002010-0x00048F` - Индексы по экранам заднего фона, уровни 3.1-3.4, все экраны (кроме водных)
  * `0x002790-0x00290F` - Индексы по экранам заднего фона, уровень 3.0, все водные экраны
  * `0x002910-0x002A8F` - Индексы по экранам заднего фона, уровни 3.1-3.4, все водные экраны
  * `0x002A90-0x00368F` - Индексы по экранам заднего фона, уровень 2 (снаружи)
  * `0x003690-0x003A07` - Таблица атрибутов заднего фона, уровень 2 (снаружи)
  * `0x003A08-0x003DC7` - Массив метатайлов, уровень 2 (снаружи)
  * `0x003DC8-0x003E0F` - Не используется
  * `0x003E10-0x00400F` - Массив метатайлов, уровень 2 (внутри) и водные экраны
* Банк 01, страница 1
  * `0x004010-0x00408F` - Массив метатайлов, уровень 2 (внутри) и водные экраны (продолжение)
  * `0x004090-0x00498F` - Индексы по экранам заднего фона, уровень 2 (внутри), все экраны (кроме водных)
  * `0x004990-0x004C8F` - Индексы по экранам заднего фона, уровень 2 (внутри), все водные экраны
  * `0x004C90-0x004ECF` - Таблица атрибутов заднего фона, уровень 2 (внутри), все экраны (кроме водных)
  * `0x004ED0-0x004FAF` - Таблица атрибутов заднего фона, уровень 2 (внутри), все водные экраны
  * `0x004FB0-0x00500F` - Не используется
  * `0x005010-0x005C0F` - Индексы по экранам заднего фона, уровень 1.0, все экраны
  * `0x005C10-0x005F8F` - Таблица атрибутов заднего фона, уровень 1.0, все экраны
  * `0x005F90-0x00600F` - Не используется
* Банк 01, страница 2
  * `0x006010-0x0062C7` - Массив метатайлов, уровень 1.0
  * `0x0062C8-0x006A47` - Индексы по экранам заднего фона, уровни 1.1-1.3, все экраны
  * `0x006A48-0x006C77` - Таблица атрибутов заднего фона, уровни 1.1-1.3, все экраны
  * `0x006C78-0x006EA7` - Массив метатайлов, уровни 1.1-1.3
  * `0x006EA8-0x00700F` - Не используется
  * `0x007010-0x00710B` - Таблица по комнатам, каждый элемент которого представляет собой адрес таблицы, содержащей значения столкновений
  * `0x00710C-0x007207` - Таблица по комнатам, каждый элемент которого представляет собой адрес таблицы, содержащей значения цветовых палитр заднего фона
  * `0x007208-0x0073C7` - Список цветовых палитр заднего фона в рамках отдельной комнаты
  * `0x0073C8-0x007CE5` - Список таблиц со значениями столкновений, всех уровней, кроме 4 уровня
  * `0x007CE6-0x007CEF` - Не используется
  * `0x007CF0-0x007E35` - Список таблиц со значениями столкновений для 4 уровня
  * `0x007E36-0x007E37` - Не используется
  * `0x007E38-0x007FB7` - Индексы по экранам заднего фона, ездовой уровень, все экраны
  * `0x007FB8-0x00800F` - Не используется
* Банк 02, страница 1
  * `0x008010-0x008193` - Список адресов на звуковые треки, с указанием звукового канала и номером звуковой партии
  * `0x008194-0x008D73` - Список всех зашифрованных звуковых треков
  * `0x008D90-0x0093C9` - Общий механизм воспроизведения звуков и музыки в игре, используя звуковые дорожки
  * `0x0093CA-0x00940F` - Не используется
  * `0x009410-0x00949F` - Таблица адресов на диалоги персонажей в режиме "Выбор персонажа" в момент начала уровня, в момент использования радио или в момент гибели одного из них
  * `0x0094A0-0x0094A1` - Адрес на таблицу масок по возможности выпадения игровых элементов в воде
  * `0x0094A2-0x009533` - Таблица по комнатам, каждый элемент которого представляет собой адрес на информацию, необходимую для перехода в водные комнаты из текущей комнаты
  * `0x009534-0x0095AF` - Таблица по водным комнатам, каждый элемент которого представляет собой адрес на информацию, необходимую для выхода из текущей водной комнаты
  * `0x0095B0-0x0095CF` - Таблица игровых элементов, выпадаемых в воде
  * `0x0095D0-0x0095EF` - Таблица вертикальных позиций игровых элементов, выпадаемых в воде
  * `0x0095F0-0x009629` - Таблица с очками по списку врагов, который игрок получает после их ликвидации
  * `0x00962A-0x0096A1` - Таблица по водным комнатам, каждый элемент которого представляет собой адрес на комнаты, в которых игрок может провалиться
  * `0x0096A2-0x0096FA` - Список с наборами комнат, в которых игрок может провалиться
  * `0x0096FB-0x0099F6` - Список с наборами структур, необходимых для перехода в водные комнаты из текущей комнаты
  * `0x0099F7-0x009A72` - Таблицу масок по возможности выпадения игровых элементов в воде
  * `0x009A73-0x009AF8` - Список с наборами структур, необходимых для выхода из текущей водной комнаты
  * `0x009AF9-0x009EF4` - Список с наборами плиток для диалогов персонажей в режиме "Выбор персонажа" в момент начала уровня, в момент использования радио или в момент гибели одного из них
  * `0x009EF5-0x00A00F` - Не используется
* Банк 02, страница 2
  * `0x00A010-0x00A02B` - Таблица адресов на изображения Дзэнигаты
  * `0x00A02C-0x00A047` - Таблица адресов на изображения водолаза из водных комнат
  * `0x00A048-0x00A1C5` - Список изображений водолаза из водных комнат
  * `0x00A1C6-0x00A393` - Список изображений Дзэнигаты
  * `0x00A394-0x00A40F` - Не используется
  * `0x00A410-0x00A69B` - Массив метатайлов, уровень 4
  * `0x00A69C-0x00A69F` - Не используется
  * `0x00A6A0-0x00B29F` - Индексы по экранам заднего фона, ездовой уровень, все экраны
  * `0x00B2A0-0x00B68F` - Таблица атрибутов заднего фона, ездовой уровень, все экраны
  * `0x00B690-0x00B9BB` - Массив метатайлов, ездовой уровень
  * `0x00B9BC-0x00B9BF` - Не используется
  * `0x00B9C0-0x00BA7F` - Индексы по экранам заднего фона, уникальная комната уровней 3.1-3.4
  * `0x00BA80-0x00BAB7` - Таблица атрибутов заднего фона, уникальная комната уровней 3.1-3.4
  * `0x00BAB8-0x00BC37` - Индексы по экранам заднего фона, уникальная комната уровня 2 (первая и вторая половина)
  * `0x00BC38-0x00BCA7` - Таблица атрибутов заднего фона, уникальная комната уровня 2 (первая и вторая половина)
  * `0x00BCA8-0x00BD67` - Индексы по экранам заднего фона, уникальная комната уровней 1.1-1.3
  * `0x00BD68-0x00BD9F` - Таблица атрибутов заднего фона, уникальная комната уровней 1.1-1.3
  * `0x00BDA0-0x00BE5F` - Индексы по экранам заднего фона, уникальная комната уровня 4
  * `0x00BE60-0x00BF1F` - Индексы по экранам заднего фона, комната с боссом уровня 4
  * `0x00BF20-0x00BF57` - Таблица атрибутов заднего фона, уникальная комната уровня 4
  * `0x00BF58-0x00BF97` - Таблица атрибутов заднего фона, комната с боссом уровня 4
  * `0x00BF98-0x00C00F` - Не используется
* Банк 03, страница 1
  * `0x00C010-0x00D68F` - Индексы по экранам заднего фона, уровень 4, все экраны (кроме водных)
  * `0x00D690-0x00D80F` - Индексы по экранам заднего фона, уровень 4, все водные экраны
  * `0x00D810-0x00D87F` - Таблица атрибутов заднего фона, уровень 4, все водные экраны
  * `0x00D880-0x00DF0F` - Таблица атрибутов заднего фона, уровень 4, все экраны (кроме водных)
  * `0x00DF10-0x00E00F` - Не используется
* Банк 03, страница 2
  * `0x00E010-0x00FE4A` - Логика поведения всех боссов и некоторых врагов (скелет, мумия, рыцарь в доспехах и др.)
  * `0x00FE4B-0x01000F` - Не используется
* Банк 04, страница 1
  * `0x010010-0x010019` - Список адресов на структуры для создания анимации выстрела на главном меню игры
  * `0x01001A-0x01001B` - Адрес на таблицу контрольных точек игры по комнатам
  * `0x01001C-0x01001D` - Адрес на таблицу палитр каждого врага из игры
  * `0x01001E-0x010023` - Таблица адресов на имитацию поведения каждого персонажа в игре в демонстрационных сценах
  * `0x010024-0x010029` - Список номеров банков CHR ROM для режима выбора персонажа
  * `0x01002A-0x010039` - Список цветовых палитр для отображения Клариссы в кат-сценах
  * `0x01003A-0x010051` - Список плиток для copyright
  * `0x010052-0x01006D` - Список цветовых палитр для отображения игровых и неигровых персонажей в демонстрационных сценах
  * `0x01006E-0x0100CD` - Список структур для отображения игровых и неигровых персонажей в демонстрационных сценах (в первой и последней кат-сцене с Клариссой)
  * `0x0100CE-0x0100D4` - Список номеров банков CHR ROM для главного меню
  * `0x0100D4-0x0100F3` - Список цветовых палитр для отображения главного меню
  * `0x0100F4-0x010148` - Список структур для отображения главного меню
  * `0x010149-0x010170` - Список структур для отображения экрана выбора персонажа
  * `0x010171-0x010189` - Список структур для отображения экрана после гибели всех персонажей
  * `0x01018A-0x0101A1` - Список цветовых палитр для отображения экрана выбора персонажа
  * `0x0101A2-0x0101A9` - Не используется
  * `0x0101AA-0x0102A5` - Список адресов по комнатам на таблицы контрольных точек
  * `0x0102A6-0x0103A1` - Список адресов по комнатам на таблицы разрушаемых стен (в которые потом можно войти как в комнату)
  * `0x0103A2-0x0103CD` - Список адресов на структуры для отображения закрытых и открытых дверей
  * `0x0103CE-0x0104C9` - Список адресов по комнатам на таблицы контрольных точек, используемых после гибели или ареста персонажа
  * `0x0104CA-0x010547` - Таблица с номерами врагов по комнатам, которые появляются, когда длительное время других врагов нет на экране
  * `0x010548-0x0105C5` - Таблица размеров каждой комнаты в экранных единицах
  * `0x0105C6-0x0105CD` - Список адресов по таблицы контрольных точек для уникальных комнат
  * `0x0105CE-0x0105DD` - Таблица атрибутов для отображения панели с сообщениями неигровых персонажей
  * `0x0105DE-0x010636` - Таблица склонов на крышах из уровня 1.0
  * `0x010637-0x010666` - Таблица с различной информацией по отображения машины и персонажа в ездовом уровне
  * `0x010667-0x010684` - Таблица с информацией по дорожным холмам в ездовом уровне
  * `0x010685-0x010698` - Список таблиц контрольных точек для уникальных комнат
  * `0x010699-0x0106D8` - Таблица атрибутов для отображения экрана выбора персонажа
  * `0x0106D9-0x010708` - Список плиток используемых для отображения экрана выбора персонажа
  * `0x010709-0x01079E` - Список плиток используемых для отображения игровых и неигровых персонажей в демонстрационных сценах (в первой с Клариссой)
  * `0x01079F-0x0107AC` - Таблица атрибутов используемых для отображения игровых и неигровых персонажей в демонстрационных сценах (в первой с Клариссой)
  * `0x0107AD-0x0107F4` - Список плиток используемых для отображения последней кат-сцены с Клариссой
  * `0x0107F5-0x0107FA` - Таблица атрибутов используемых для отображения последней кат-сцены с Клариссой
  * `0x0107FB-0x01092C` - Список плиток и таблицы атрибутов используемых для отображения главного меню
  * `0x01092D-0x01094A` - Таблица с информацией по имитации поведения персонажа в игре в демонстрационных сценах
  * `0x01094B-0x010D46` - Таблица контрольных точек (первичный список)
  * `0x010D47-0x010D4A` - Не используется
  * `0x010D4B-0x010F12` - Таблица контрольных точек (вторичный список)
  * `0x010F13-0x011A3D` - Список таблиц контрольных точек по каждой комнате
  * `0x011A3E-0x011ABE` - Список таблиц разрушаемых стен (в которые потом можно войти как в комнату) по каждой комнате
  * `0x011ABF-0x011B9C` - Список таблиц контрольных точек, используемых после гибели или ареста персонажа, по каждой комнате
  * `0x011B9D-0x011CDC` - Список структур для отображения закрытых и открытых дверей
  * `0x011CDD-0x011D8A` - Таблица палитр каждого врага из игры
  * `0x011D8B-0x011E3C` - Список структур для создания анимации выстрела на главном меню игры
  * `0x011E3D-0x01200F` - Не используется
* Банк 04, страница 2
  * `0x012010-0x01210B` - Список адресов из графической памяти для отображения сообщений в диалогах с неиграбельными персонажами
  * `0x01210C-0x01210F` - Не используется
  ....
* Банк 05, страница 1
  * `0x014010-0x0140D9` - Список адресов на структуры для спрайтов
  * `0x0140DA-0x01410F` - Не используется
  * `0x014110-0x0141CF` - Список адресов на структуры для спрайтов (продолжение)
  * `0x0141D0-0x01420F` - Не используется
  * `0x014210-0x0142F7` - Список адресов на структуры для спрайтов (продолжение)
  * `0x0142F8-0x015EF0` - Список структур для спрайтов (количество тайлов и список атрибутов спрайтов)
  * `0x015EF1-0x01600F` - Не используется
